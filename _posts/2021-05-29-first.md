배열 (Array) 2

배열로 할수 있는 것들 : 요소(element)를 추가 할수 있다.

```javascript

 let  yourNumber = [11, 22, 33, 44, 55] ; 
 yourNumber.push(66);

// push를 집어넣고 괄호안에 (66)를 집어넣게 되면 
// 배열 끝에 인덱스와 그 요소 66가 추가 된다 .

// yourNumber
// (6) [11, 22, 33, 44, 55, 66]
```

배열로 할수 있는 것들 2 : 요소를 삭제 할수 있다. 

```javascript

 let  yourNumber = [11, 22, 33, 44, 55] ; 
yourNumber.pop();  // pop를 사용하면 배열 끝 인덱스와 요소 삭제.

// yourNumber
(5) [11, 22, 33, 44, 55]
```
